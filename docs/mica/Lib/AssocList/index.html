val lex_compare_pair : (Base.int * Base.string) -> (Base.int * Base.string) -> Base.int

Polymorphic lexicographic comparison for pair types 'a * 'b, using the default compare functions for types 'a and 'b. For two pairs (a1, b1), (a2, b2):

  • If compare a1 a2 < 0, lex_compare_pair (a1, _) (a2, _) < 0
  • If compare a1 a2 > 0, lex_compare_pair (a1, _) (a2, _) > 0
  • If compare a1 a2 = 0, lex_compare_pair (a1, b1) (a2, b2) = compare b1 b2
  • Adapted from https://stackoverflow.com/a/20348513
